# 二、品牌命名理念

SEEFOOD 靈感來自：

```txt
SEAFOOD（海鮮）
→ SEEFOOD（看見食物）

```

透過諧音與雙關語，  
建立具有記憶點與現代感的品牌名稱。

品牌希望傳達：

- 看見食材真正的樣子
- 看見海鮮的新鮮
- 看見生活中的儀式感
- 重新定義海鮮電商

---

# 三、品牌定位

## 品牌定位

現代化高品質海鮮生活品牌

不是：

- 傳統魚市場
- 傳統水產行
- 廉價團購海鮮

而是：

```txt
Premium Seafood Lifestyle Brand

```
